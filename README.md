# Blp Convert

Simple web based BLP converter created using [haax-blp](https://github.com/Haaxor1689/haax-blp). You can find it hosted at:

### https://blp-convert.haaxor1689.dev/

## Features

- Load BLP or PNG image
- Preview the image and its settings
- Save to PNG
- Save to BLP
  - DXT1/DXT3/DXT5 compression
  - Palette compression with automatic image quantization
  - Automatic alpha flattening to selected alpha depth
  - MipMap generation
  - Also supports uncompressed BLP output
